Room Booking
Service

Smart meeting room reservation system

Kamil Akhmadiev k.akhmadiev@innopolis.university DSAI-02

The Problem

End Users
Company employees who need to reserve meeting rooms and office administrators who manage them
Problem
Manual booking leads to double bookings, scheduling conflicts, and wasted time coordinating who uses which room
Solution
A web app where admins set schedules, the system auto-generates time slots, and employees book rooms with real-time availability

Implementation

Version 1
  • Room & schedule management (admin)
  • Automatic 30-min slot generation
  • Visual timeline with drag-to-select
  • Go backend + PostgreSQL + React
Version 2
  • Show who booked each slot
  • User registration & login (JWT)
  • Conference link generation
  • Enriched API + Docker deploy
TA Feedback Addressed
